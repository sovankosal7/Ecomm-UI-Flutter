# 🛍️ Modern E-Commerce UI - Flutter Mobile App

A high-fidelity, responsive E-commerce mobile application UI built using Flutter. This project showcases clean UI/UX design, custom widget architecture, and smooth navigation flows.

## 📱 Project Overview
This repository contains the frontend implementation of a modern shopping experience. The focus was on creating a highly interactive and user-friendly interface that could easily be connected to a Laravel or Firebase backend.

## ✨ Key Features
* **Custom Navigation:** Smooth transitions between categories and product screens.
* **Responsive Layout:** Optimized for various screen sizes using Flutter's layout widgets.
* **Component-Based UI:** Reusable widgets for product cards, buttons, and input fields.
* **Stateful Interactions:** Interactive elements for product selection (Size/Color) and cart management.

## 🛠️ Tech Stack
* **Framework:** Flutter (Dart)
* **Design Pattern:** Clean Architecture (UI Layer)
* **Assets:** Custom SVG icons and Google Fonts (Poppins/Roboto)
